    事件类型。
  • fn : Function
    监听函数。
  • scope : Object Optional
    作用域。指定监听函数fn中的this指针所指向的对象。默认为undefined。

Parameters

    事件类型。
  • params : Object Optional
    附带参数。
  • bubble : Boolean Optional

  • Boolean
    如果成功调度了事件，则值为 true。 值 false 表示失败或对事件调用了 preventDefault(). 仅当useBrowserEvent为false时有效。
