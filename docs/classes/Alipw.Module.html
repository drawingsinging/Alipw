Class Alipw.Module

Extends Alipw.Component.
Inheritance Module > Component
Defined in: Alipw.Module.js.
Author: zhangwen.cao[zhangwen.cao@aliyun-inc.com].
模块。模块是一个独立个体。它可以独立的调用一个远程页面，并在指定的位置进行渲染。同时，它还可以调用指定的JavaScript文件并在该模块的资源完全加载完毕后执行。模块还可对在其内部创建的可视控件进行资源管理，在自身被销毁时，通过该模块创建的可视控件也将被销毁。
